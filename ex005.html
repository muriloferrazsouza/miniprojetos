Aprendendo DOM

DOM significa Document Object Model e só está presente quando rodando JavaScript no navegador

Iremos fazer a árvore DOM do site

Basicamente, o DOM é o que dita como o código em JavaScript será escrito (WINDOW . DOCUMENT . BODY)

Cada palavra antes do ponto se refere a sua poisção hierárquica no site, em que se começa pelo WINDOW, que se refere á página como um todo, passando para DOCUMENT, que é todo o arquivo que representa a página, seguindo para BODY (ou HEAD), e assim por diante

Quando se tem multiplos objetos com mesma tag (como por exemplo esses parágrafos dessa página, em que todos são 'p'), no script é usado o comando 'getElementsByTagName()' para selecionar o correto (sempre lembrando de usar caixa alta) e a contagem é feita a partir do 0 (0 representa o primeiro parágrafo).

Também é usado, para facilitar, o 'getElementsById()', que se referencia a QUALQUER tag do HTML, porém você, como usuário, precisa nomear uma TAG ao elemento, como foi feito com a DIV abaixo

Alem dessas duas formas, ainda existem o 'getElementsByName()' e 'getElementsByClassName()', que segue a mesma lógica do ID, basta você nomear o elemento no HTML, para puxá-lo usando o JavaScript
