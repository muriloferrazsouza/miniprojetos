Existem diversos eventos de DOM, que podem ser feitos. Para citar alguns exemplos, podem ocorrer alguns apenas com movimentos de mouse. Caso o mouse entre em uma DIV que tenha esses eventos configurados, pode ocorrer o 'mouseenter', ou para sair o 'mouseout', o clique do mouse 'click', e o mexer do mouse 'mousemove'

Para uma lista de todos os eventos possíveis, pesquise no google por 'JavaScript DOM event list' ou clique aqui

Para tratar esses eventos, é necessário a utilização de funções. Para usar funções, você cria um bloco de código (que pode ter qualquer número de linhas), e o coloca dentro de colchetes. Você nomea esse colchete como 'function' seguida de seu nome, como exemplificado na seção de script desta página

Em contrapartida, também é possivel fazer sem nomeá-los diretamente na div, como feito no primeiro quadrado, pois existe uma função no JavaScript chamada de 'EventListener', e ele é configurado diretamente no script, como visto para o segundo quadrado

Caso não funcione, lembre-se de verificar no 'Inspecionar Elemento', na parte do console do navegador, para ver se encontra a referência de onde possa estar o erro.
